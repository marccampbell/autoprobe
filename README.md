# autoprobe

AI-powered performance optimization that actually fixes your code.

Point it at a slow endpoint, set your constraints, and let it iterate on improvements — rewriting queries, suggesting indexes, optimizing algorithms — until your code is faster.

## Install

```bash
go install github.com/marccampbell/autoprobe@latest
```

## Usage

```bash
autoprobe --help
```
